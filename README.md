# iPath.NET

This is a port of the 20y old iPath System written in PHP to a modern ASP.net core application.

The original source code is available at https://sourceforge.net/projects/ipath/

The goal of this effort is to replace the system on http://ipath-network.com

## Design
 - Frontend: Blazor
 - ORM: EF-Core

At the moment this is purely a playground ...
